# Neonque Futuristic Frontend Starter

Production-oriented mobile-first starter architecture for a futuristic automotive + fabrication showcase website.

## Stack
- Next.js 15
- React
- Tailwind CSS
- Framer Motion
- TypeScript

## Install
```bash
npm install
npm run dev
```

## Core Features
- Cyberpunk UI foundation
- Mobile-first layout
- Floating WhatsApp CTA
- Animated hero section
- Project showcase cards
- Glassmorphism design system
- Dark neon palette
